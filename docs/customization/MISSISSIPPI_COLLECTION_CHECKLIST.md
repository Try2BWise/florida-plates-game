# Mississippi Collection Checklist

Use this as a quick tracking aid while you gather data and images.

## Source Hunting

- official Mississippi plate listing page
- official Mississippi brochure PDF if available
- official Mississippi motorcycle plate listing
- university plate pages or PDFs
- military / veterans plate pages
- any public service / law enforcement / firefighter plates
- any archived or alternate official image sources if the main site is incomplete

## For Each Plate

- official name
- display-friendly name
- whether it is current or legacy
- whether it is active or discontinued
- category
- beneficiary / sponsor
- eligibility notes if relevant
- image URL
- planned local filename
- useful search terms

## Variant Review

Check whether any of these exist:

- current and legacy versions
- motorcycle variants
- wheelchair / accessibility variants
- standard plate design changes that should be separate
- duplicate rows that are really the same logical plate

## Search-Term Ideas To Capture Early

- abbreviations
- mascots
- school nicknames
- pro team names
- colors
- animals
- obvious visual symbols
- likely user phrases

## Image Prep Notes

- prefer one clean canonical image per logical or true variant plate
- save a stable local filename plan early
- keep a record of the source URL even if you download locally
- do not worry yet about bundling all images into the app shell

## Architecture Reminder

For future multi-state support, Mississippi should eventually provide:

- a master dataset
- a generated runtime plate driver
- state branding assets
- optional badge definitions
- optional geography definitions

The current goal is just to get the plate data and images into good shape first.
